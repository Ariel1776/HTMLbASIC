Criando Tabelas

Para criar uma tabela precisamos de Linhas e Colunas, as mesmas são representadas pelas sintaxes abaixo

"< table > Responsável por criar a TABELA < / table >"

"< td > Responsável por criar a Célula < / td >"

"< tr > Responsável por criar a as Linhas < / tr >"
