Atribuindo - ACTION

Com o atributo ACTION, podemos especificar a URL do script no qual será enviado as informações"< form action = "EMAIL ou LINK" > CONTEÚDO < / form >"

Exemplo:

Primeiro formulário

Atribuindo - METHOD

Com o atributo METHOD, podemos especificar a maneira de envio dos dados para o servidor Web"< form method=" get _ post" > CONTEÚDO < / form >"

  • get - Utilizado para respostas únicas;
  • post - Utilizado por permitir a transmissão de uma maior quantidade ao servidor; Envia os dados ao servidor
